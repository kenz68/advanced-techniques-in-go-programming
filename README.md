# Advanced Techniques in Go Programming

A hands-on course for intermediate Go developers and experienced programmers from other languages who want to deepen their Go expertise.

## Course Overview

This course covers advanced Go programming topics through a combination of theory and practical exercises. By the end of the course, you will be able to build production-quality services and tools using idiomatic Go.

## Topics Covered

| Lesson | Topic |
|--------|-------|
| 01 | Course Introduction & Objectives |
| 02 | RESTful Services in Go |
| 03 | WebSocket Protocols |
| 04 | Generics in Go |
| 05 | Practical Utility Development |

## Prerequisites

- Basic familiarity with Go syntax (variables, functions, structs, interfaces)
- Experience with at least one other programming language
- Go 1.21 or later installed

## Getting Started

```bash
# Clone the repository
git clone https://github.com/kenz68/advanced-techniques-in-go-programming.git
cd advanced-techniques-in-go-programming

# Navigate to a lesson and run the examples
cd lesson01
go run main.go
```

## Course Structure

Each lesson directory contains:
- `README.md` — lesson objectives and key concepts
- `main.go` — runnable example demonstrating the lesson material
- Exercise prompts where applicable
